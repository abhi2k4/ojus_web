The Athletic events For OJUS 2023

Rules

1.On the starters whistle all sprinters begin the race.
2.Sprinters must not start running before starters whistle, if done so it would be give false start.
3.Sprinters should not cross the lane allotted to them, if done so sprinters will be disqualified.
4.Stepping on the white line of the lane is also considered crossing the lane & the sprinter would be disqualified.
